# jhansi
A procedural, statically typed systems programming language.

🚧 **v0.1.0 in active development** — targeting a working `hello world` via C codegen.  

## Goals
- Procedural, Go-inspired syntax
- Native compilation via C codegen (initial backend)
- Explicit memory management without a GC
